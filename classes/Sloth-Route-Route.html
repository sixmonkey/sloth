Router

Handles HTTP routing for the Sloth framework using FastRoute. Provides methods for defining routes and dispatching requests to appropriate controllers.

Tags
since
1.0.0
see
https://github.com/nikic/FastRoute

FastRoute library

example 
Route::get('/about', ['controller' => 'PageController', 'action' => 'about']);
Route::post('/contact', ['controller' => 'ContactController', 'action' => 'submit']);
Route::any('/api/{id}', ['controller' => 'ApiController']);
