MenuItem.php : 44

Menu Item Model

Represents a WordPress navigation menu item with support for various item types (posts, pages, categories, custom links).

Tags
since
1.0.0
extends
example 
// Get menu by location
$menu = Menu::location('primary');

// Iterate through items
foreach ($menu->items as $item) {
    echo $item->title;
    echo $item->url;
    echo $item->current ? 'active' : '';
}
