Handles calls to undefined methods.

public __call(string $method, array<int, mixed> $parameters) : mixed|null

This magic method catches any calls to undefined methods. The 'boot' method is silently ignored to allow for lazy booting. All other undefined method calls throw an exception.

Parameters
$method : string

The method name that was called

$parameters : array<int, mixed>

The parameters passed to the method

since

throws
Exception

If the method is not 'boot' and is not defined

example

// Throws exception

Returns null for boot(), or throws exception
