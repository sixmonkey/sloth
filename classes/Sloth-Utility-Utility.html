Utility.php : 22

Utility class with helper methods for name conversion.

This class provides static methods for converting names between different naming conventions used throughout the Sloth framework:

  • Module naming (PascalCase with Module suffix)
  • View naming (kebab-case)
  • ACF naming (snake_case with optional prefix)
  • Normalized names (cleaned for comparison)

public static acfize(string $name[, bool $prefixed = true ]) : string

Transforms a name into snake_case suitable for ACF field group keys. Optionally adds the 'group_module_' prefix used by Sloth for module-specific field groups.

Parameters
$name : string

The name to convert (e.g., 'hero_image', 'HeroImage')

$prefixed : bool = true

Whether to add the 'group_module_' prefix

Tags
since
1.0.0
example 
Utility::acfize('hero_image');        // 'group_module_hero_image'
Utility::acfize('HeroImage');          // 'group_module_hero_image'
Utility::acfize('my_field', false);   // 'my_field'
Utility::acfize('featured_posts');     // 'group_module_featured_posts'
